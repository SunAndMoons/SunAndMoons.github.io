opendrive

1. 前言

ASAM OpenDrive 描述驾驶仿真应用所需的静态道路交通网络（路网），并提供了标准交换格式的说明文档。

对道路以及道路上的物体进行表述，道路、查到、交叉路口等内容进行建模。

使用文件扩展名为 .xord xml文件作为描述路网的基础。通过 node 被构建，用户可自定义数据扩展结点。

1.1 单位

若没有特殊说明，所有数据采用SI单位：

  • length: [m]
  • angle:[rad]
  • time:[s]
  • speed: [m/s]
1.2 曲率

​ 曲率= 1/曲率半径

  • 正曲率：左曲线（逆时针运动）
  • 负曲率：右曲线（顺时针运动）

2. 通用架构

2.1 文件结构

数据存储在XML文件中，文件扩展名为 .xord， 压缩文件的扩展名为 .xordz。

2.2 头文件
名称 类型 单位 值 描述
revMajor ushort - 1 格式主版本号
revMinor ushort - 4 格式次版本号
name string - - 数据库名称
version string - - 本路网的版本号
date string - - 数据库创建的时间
north double m - 最大惯性y值
south double m - 最小惯性y值
east double m - 最大惯性x值
west double m - 最小惯性x值
vendor string - - 开发商名称

3. 坐标系

OPENDRIVE 使用三种类型的坐标系。

  • 惯性x/y/z轴坐标系

  • 参考线s/t/h轴坐标系

  • 局部u/v/z轴坐标系
